
Library Management System - Java + MySQL

1. Import db.sql into your MySQL server.
2. Update DB credentials in LibraryDAO.java.
3. Compile all Java files from the src folder.
4. Add MySQL JDBC connector JAR to your classpath.
5. Run LibraryApp.java to start the app.

Dependencies:
- Java 8 or higher
- MySQL Server
- mysql-connector-java-X.X.X.jar (download from: https://dev.mysql.com/downloads/connector/j/)
